# JavaScript
>[!note]
>### Определение
> **JavaScript** — это то, что превращает статичную страницу (как картинка) в интерактивную (можно нажимать, двигать, обновлять без перезагрузки).

### Что он может делать?
```js
// 1. изменить текст на странице
document.getElementById("title").innerText = "новый заголовок";

// 2. реагировать на клик
button.addEventListener("click", function() {
    alert("ты нажал на кнопку");
});

// 3. обновлять данные без перезагрузки
// (как лайки в соцсетях - сразу меняется число)

// 4. анимации и движение
element.style.transform = "rotate(45deg)";

// 5. работа с данными
const user = { name: "Иван", age: 25 };
```

### Время для базового синтаксиса (тобиш, основы)
```js
// Переменные
let name = "Иван";      // можно менять
const age = 25;         // нельзя менять (константа)

// Функции
function sayHello(person) {
    return "Привет, " + person + "!";
}
sayHello("Анна");  // "Привет, Анна!"

// Стрелочные функции (современный стиль)
const add = (a, b) => a + b;
add(5, 3);  // 8

// Условия
if (age >= 18) {
    console.log("Совершеннолетний");
} else {
    console.log("Несовершеннолетний");
}

// Циклы
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
}
```
# HTML
>[!note]
>### Определение
>**HTML** - то не язык программирования, а **язык разметки**. Он отвечает за структуру страницы: что где находится (заголовок, текст, картинка, кнопка).

### Обычный пример страницы:
```html
<!DOCTYPE html>  <!-- это HTML5 документ -->
<html>           <!-- начало всей страницы -->
<head>           <!-- служебная информация -->
    <title>Мой сайт</title>  <!-- название вкладки -->
</head>
<body>           <!-- видимая часть сайта -->
    <h1>Привет, мир!</h1>      <!-- главный заголовок -->
    <p>Это мой первый сайт.</p> <!-- параграф текста -->
    <img src="питергриффин.jpg">   <!-- картинка -->
    <button>Нажми меня</button> <!-- кнопка -->
</body>
</html>
```
>[!note]
>### DOCTYPE
> **DOCTYPE** - это "паспорт" или "указание версии" для сайта.
> 
> **Проще говоря** это первая строка в HTML-документе, которая говорит браузеру: "Эй, это документ HTML5, обрабатывай его по современным правилам!"

### Зачем это нужно?
Без DOCTYPE: Браузер думает "старый сайт 90-х" → включает режим совместимости (quirks mode)

С DOCTYPE: Браузер думает "современный сайт" → включает стандартный режим (standards mode)

# CSS
>[!note]
> ### Определение
> CSS — это "одежда" или "косметика" для **HTML**
> ### Проще говоря:
> **CSS** — это язык, который делает **HTML** красивым: задаёт цвета, шрифты, отступы, анимации. Если **HTML** — это скелет человека, то **CSS** — это его одежда, причёска и макияж.

```css
.title {
    color: blue;           /* Синий цвет */
    font-size: 32px;       /* Большой шрифт */
    text-align: center;    /* По центру */
}

.text {
    color: gray;           /* Серый цвет */
    font-family: Arial;    /* Шрифт Arial */
    line-height: 1.5;      /* Межстрочный интервал */
}

.btn {
    background: red;       /* Красный фон */
    color: white;          /* Белый текст */
    padding: 10px 20px;    /* Отступы внутри */
    border-radius: 5px;    /* Закруглённые углы */
    border: none;          /* Без рамки */
}
```

текст
```css
.text {
    font-family: "Arial", sans-serif; /* Шрифт */
    font-size: 16px;                  /* Размер */
    font-weight: bold;                /* Жирность */
    text-align: center;               /* Выравнивание */
    text-decoration: underline;       /* Подчёркивание */
}
```
блоки и отступы
```css
.box {
    width: 200px;           /* Ширина */
    height: 100px;          /* Высота */
    padding: 20px;          /* Внутренний отступ */
    margin: 10px;           /* Внешний отступ */
    border: 2px solid black;/* Рамка */
}
```
анимации
```css
@keyframes slide {
    from { transform: translateX(0); }
    to { transform: translateX(100px); }
}

.element {
    animation: slide 2s infinite;
}
```

### Итог:
**CSS** = "Волшебная кисточка, которая превращает скучный чёрно-белый **HTML** в красочный, современный сайт"

### Если это строительство дома(пример):

**HTML** = фундамент, стены, окна (что строим)

**CSS** = краска, обои, мебель (как выглядит)

**JavaScript** = электричество, водопровод (что умеет)
